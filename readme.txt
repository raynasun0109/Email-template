## How to test its responsive on mobile version
1. Opened "inspect" element toolkit in some mainstream browsers like Chrome, FireFox, Safari
2. Selected different popular phone devices to check whether the page is responsive or not
3. In FireFox browser, I also checked how the page looks like when user rotated their phone
4. Then I adjusted the viewport to different width to make sure tha the page is responsive with other screen sizes
